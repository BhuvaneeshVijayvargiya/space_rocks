# Space_Rocks
It is an arcade game based-off classic asteroids game. <br>
You can move using WASD, and shoot using Left Mouse Button. <br>
Contains different rooms with different playstyles. (contains secret room) <br>
It has two cheat codes, P and R. (custom message for when using cheats)<br>
Difficulty increases with time, by increasing the number of rocks everytime you shoot them.
